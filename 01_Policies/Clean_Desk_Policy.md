# Clean Desk and Clear Screen Policy

**Document Version:** 1.0  
**Effective Date:** December 2025  
**Owner:** EcoMetricx InfoSec Team (Andrei Ionete - andrei@ecometricx.com)  
**Review Cycle:** Annually

## 1. Purpose

Establish requirements for protecting sensitive information from unauthorized access by maintaining clean workspaces and secure screens.

## 2. Scope

Applies to all EcoMetricx employees, contractors, and visitors in all work environments including remote work locations.

## 3. Clean Desk Requirements

### 3.1 End of Day

- Lock away all sensitive documents
- Secure removable media (USB drives, external drives)
- Clear whiteboards of sensitive information
- Dispose of sensitive waste in shred bins

### 3.2 During Absences

- Lock workstation before leaving desk
- Secure sensitive documents in drawers
- Do not leave passwords written down

### 3.3 Printing

- Retrieve printed documents immediately
- Use secure print release if available
- Shred unwanted prints containing sensitive data

## 4. Clear Screen Requirements

### 4.1 Screen Lock

- Lock screen when leaving workstation (Windows: Win+L, Mac: Ctrl+Cmd+Q)
- Automatic lock after 5 minutes of inactivity (required configuration)

### 4.2 Screen Privacy

- Position monitors away from public view
- Use privacy screens in shared spaces
- Close sensitive applications before screen sharing

### 4.3 Remote Work

- Same requirements apply to home offices
- Ensure family members cannot view sensitive information
- Use privacy screens on laptops in public

## 5. Information Classification

| Classification | Clean Desk | Clear Screen |
|----------------|------------|--------------|
| Public | Recommended | Required |
| Internal | Required | Required |
| Confidential | Required | Required |
| Restricted | Required + Locked | Required + Encrypted |

## 6. Compliance

- Random audits may be conducted
- Violations reported to HR
- Repeated violations may result in disciplinary action

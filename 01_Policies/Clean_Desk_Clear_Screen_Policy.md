# Clean Desk and Clear Screen Policy

**Version:** 1.0  
**Effective Date:** December 2025  
**Owner:** EcoMetricx InfoSec Team (Andrei Ionete - andrei@ecometricx.com)  
**Review Cycle:** Annually

## 1. Purpose

Establish requirements for protecting sensitive information in the workplace by ensuring clean desks and clear screens. Supports SOC 2 Confidentiality criteria and aligns with ISO 27001 A.7.7.

## 2. Scope

- All EcoMetricx employees and contractors
- All work locations (offices, home offices, remote locations)
- All company devices and workstations
- Physical and electronic documents

## 3. Clean Desk Requirements

### 3.1 End of Day

Before leaving the workspace:

- [ ] Clear all documents from desk surface
- [ ] Secure Confidential/Restricted documents in locked storage
- [ ] Remove sticky notes with sensitive information
- [ ] Store portable media (USB drives) securely
- [ ] Clear whiteboards with sensitive information

### 3.2 During Work Hours

When away from desk (meetings, breaks, etc.):

- [ ] Turn over or cover sensitive documents
- [ ] Do not leave Confidential documents unattended
- [ ] Secure Restricted documents when not actively in use

### 3.3 Document Handling by Classification

| Classification | Desk Handling | Storage |
|---------------|---------------|---------|
| Public | May remain on desk | Open storage |
| Internal | Minimize visible exposure | Standard filing |
| Confidential | Do not leave unattended | Locked storage |
| Restricted | Must be secured at all times | Locked storage with access log |

## 4. Clear Screen Requirements

### 4.1 Screen Lock

- **Automatic Lock:** Screens must lock after 5 minutes of inactivity
- **Manual Lock:** Lock screen when stepping away (Windows: Win+L, Mac: Ctrl+Cmd+Q)
- **Unattended Devices:** Never leave devices unlocked and unattended

### 4.2 Configuration Requirements

All EcoMetricx devices must have:
- [ ] Screen saver enabled
- [ ] Password/PIN required to unlock
- [ ] Automatic lock after 5 minutes maximum

### 4.3 Public Locations

When working in public locations (coffee shops, airports, etc.):
- [ ] Use privacy screen filter
- [ ] Position screen away from observers
- [ ] Avoid displaying Confidential/Restricted data
- [ ] Lock screen before any absence

## 5. Printing and Copying

### 5.1 Sensitive Documents

- [ ] Retrieve printed documents immediately
- [ ] Do not leave documents on printer/copier
- [ ] Use secure print feature for Confidential/Restricted documents
- [ ] Dispose of mis-prints in secure shredding bin

### 5.2 Copy Machines

- [ ] Clear copy machine memory after scanning Confidential documents
- [ ] Remove originals immediately after copying

## 6. Meetings and Shared Spaces

### 6.1 Before Meetings

- [ ] Remove previous meeting materials from whiteboards
- [ ] Clear any sensitive documents left by others

### 6.2 After Meetings

- [ ] Erase whiteboards with sensitive information
- [ ] Collect all physical documents
- [ ] Ensure no notes left behind
- [ ] Clear conference room screens (disconnect cables)

## 7. Remote Work Considerations

### 7.1 Home Office

- [ ] Secure workspace when others may access (family, visitors)
- [ ] Store Confidential documents out of sight
- [ ] Position screen away from windows/cameras
- [ ] Use video call backgrounds to obscure workspace

### 7.2 Public/Shared Spaces

- [ ] Avoid discussing sensitive matters where overheard
- [ ] Do not access Restricted data on public WiFi without VPN
- [ ] Be aware of visual eavesdropping

## 8. Disposal

### 8.1 Physical Documents

| Classification | Disposal Method |
|---------------|-----------------|
| Public | Standard recycling |
| Internal | Standard recycling (remove identifiers) |
| Confidential | Cross-cut shredding |
| Restricted | Cross-cut shredding, certificate of destruction |

### 8.2 Electronic Media

- Portable storage: Secure wipe or physical destruction
- Hard drives: NIST 800-88 compliant wiping
- Return all company media at termination

## 9. Compliance

### 9.1 Monitoring

- Random spot checks may be conducted
- Self-assessments encouraged
- Report violations to InfoSec Team

### 9.2 Violations

Non-compliance may result in:
- Verbal reminder (first instance)
- Written warning (repeat instances)
- Disciplinary action (persistent violations)

## 10. Responsibilities

### 10.1 All Personnel

- Follow clean desk and clear screen practices
- Report lost or exposed documents
- Participate in security awareness training

### 10.2 Managers

- Ensure team compliance
- Lead by example
- Address violations promptly

### 10.3 InfoSec Team

- Monitor compliance
- Provide training materials
- Update policy as needed

## 12. Quick Reference

### Before Leaving Workspace
1. Lock screen (Win+L or Ctrl+Cmd+Q)
2. Clear sensitive documents
3. Secure Confidential/Restricted items
4. Retrieve any prints

### In Public
1. Use privacy screen
2. Lock when stepping away
3. Position screen away from others
4. Avoid Confidential work if possible
